Weather Data API

URL format: http://127.0.0.1:5000/api/v1/station/data

URL Example: http://127.0.0.1:5000/api/v1/10/1988-10-25 - shows data from a station on certain day

URL Example: http://127.0.0.1:5000/api/v1/10 - shows all data from a station

URL Example: http://127.0.0.1:5000/api/v1/year/10/1988 - shows data from a station in a certain year
